## Tech Stack

- **Runtime**: Node.js (LTS)
- **Framework**: NestJS with Fastify adapter
- **Compiler**: SWC (Rust-based, faster than tsc)
- **ORM**: Prisma
- **Database**: PostgreSQL
- **Authentication**: JWT (stateless)
- **Validation**: class-validator + class-transformer
- **API Docs**: Swagger/OpenAPI

## Prerequisites

- Node.js 18+ (LTS)
- PostgreSQL 14+
- pnpm or npm or yarn
